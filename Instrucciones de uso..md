# Instrucciones de uso.

El programa se puede descargar en su defecto copiar y pegar directamente  Matlab, sin embargo, se debe de verificar si se tienen instaladas las librerias, si cuando se reproduce el programa este no funciona, entonces se deben de seguir las indicaciones de Matlab para instalar las librerias.



El programa ya tiene los valores predeterminados de los colores extraidos de la lampara de mercurio utilizada (HgW) por medio del programa de uso libre FIJI, para poder calibrar el programa, si se quiere colocar otros valores, se recomienda extraer estos valores de manera similar.




Nota: La versión utilizada para realizar el programa es la versión R2024b

## AL INICIAR

1- Una vez teniendo las librerias pertinentes, al iniciar el programa se va a pedir usar la webcam que se quiera utilizar para realizar la captura de la imagen a estudiar. Inmediantamente que se elija se abrirá una con la imagen a estudiar, este proceso puede tardar unos cuantos segundos dependiendo del procesador de la computadora a utilizar.



2- Con la imagen en una nueva pantalla, se pedirá al usuario dibujar una region de interes (ROI), se debe dejar apretado el click izquierdo de mouse, para poder la ROI, con doble click se comienza analizar la ROI elegida.



3-El programa arrojará, el nombre del color obtenido, su longitud de onda, así como, los valores de cada uno de los picos que se encuentren y el pico predominante.



NOTA: Sí por alguna razón no funciona la webcam y se tomó una foto con celular donde se aprecié bien la fluorescencia de las sustancia el programa puede funcionar Tambien con imagenes png y jpeg. Funciona de la misma manera como si se tuviera la captura por webcam.
